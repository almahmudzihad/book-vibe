# 📚 Book Vibe

Book Vibe is a modern web application built with **React**, designed to help users explore, manage, and track their reading journey. It allows you to add books to your **Read List** and **Wishlist**, with data stored in localStorage for persistence.

---

## 🚀 Features
- Browse and view detailed information about books
- Add books to **Read List** and **Wishlist**
- Prevent duplicate entries with validation
- Persistent storage using **localStorage**
- Responsive design with **Tailwind CSS**
- Deployed on **Netlify**

---

## 🛠️ Tech Stack
- **Frontend:** React, JavaScript, Tailwind CSS
- **State Management:** React hooks
- **Storage:** localStorage
- **Deployment:** Netlify
